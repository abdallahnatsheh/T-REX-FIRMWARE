Do not attempt to write a full open-source license text (e.g. AGPL-3.0) to a LICENSE file using the Write tool — the output gets blocked by the content filtering policy.

**Why:** Long legal boilerplate triggers the content filter. Happened twice in one session trying to write the AGPL-3.0 text.

**How to apply:** Tell the user to paste the license text into the file manually, or to copy it from the official source. The user is comfortable doing this and it takes seconds.
