# Device Info

## `info` / `inf`

```
CMD> info
```

3-page view of device information.

| Page | Content |
|------|---------|
| 1 | Chip model, cores, flash size, PSRAM, CPU frequency |
| 2 | WiFi MAC, BT MAC, IP address, WiFi RSSI |
| 3 | Battery %, SD card status, LoRa pins, GPS status |

| Key | Action |
|-----|--------|
| `l` / `a` | Next / previous page |
| `q` | Quit |
